# QuickType - Typing Speed Test

A clean, elegant typing speed test application to measure and improve your typing speed and accuracy.

## Features

- Measure your typing speed in Words Per Minute (WPM)
- Track your typing accuracy
- Real-time feedback on your typing
- Clean, minimalist design
- Responsive layout that works on all devices

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)
